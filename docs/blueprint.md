# **App Name**: InnovateSphere

## Core Features:

- Idea Feed: Display a scrollable feed of idea cards with user info, media preview, idea title, description, tags, and AI idea score.
- Idea Posting: Step-based idea upload process for video, image, or text, including details like problem, solution, target users, category, and desired feedback.
- AI Idea Analysis: Automatically generate an innovation score, market potential, uniqueness level, and improvement suggestions displayed visually.
- Real-time Collaboration Chat: Enable real-time chat for users collaborating on an idea.
- User Dashboard Analytics: Display user analytics like total ideas posted, total views, average rating, saved count, and collaboration requests.
- AI Idea Polishing: Offer users a button labeled "Analyze my idea" that acts as a tool which intelligently suggests improvements based on current innovation, market, and uniqueness metrics.
- Database: Utilize a scalable Firestore structure for users, ideas, ratings, feedback, and collaboration data.

## Style Guidelines:

- Primary color: Deep Royal Blue (#FFA500) to convey a sense of trust and innovation.
- Background color: Pure White (#FFFFFF) for a clean and modern look.
- Body and headline font: 'Inter', a sans-serif, will be used for its modern and neutral aesthetic, ensuring readability and a clean UI. This font will be applied to all text elements across the platform.
- Use rounded cards, soft shadows, smooth animations, and a bottom navigation bar for a modern, clean UI similar to Instagram, LinkedIn, and Product Hunt.
- Use a consistent set of minimalist icons for the bottom navigation bar and other interactive elements.
- Implement subtle animations for interactions like rating, saving, and sharing to enhance user experience.
