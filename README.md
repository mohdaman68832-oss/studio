
# InnovateSphere | Orange Edition

This is a NextJS + Firebase application built in Firebase Studio.

## 🔗 GitHub Integration
- **Repository**: [studio](https://github.com/mohdaman68832-oss/studio.git) 
- **How to Link & Sync (Code GitHub par kaise bhejein)**:
  1. **Push your code**: IDE ke left side mein **Source Control icon** (jo branch jaisa dikhta hai) par click kijiye.
  2. Wahan ek message likhiye (jaise "Initial code") aur **Commit & Push** par click kijiye. Isse code GitHub par dikhne lagega.
  3. Go to your [Firebase Console](https://console.firebase.google.com/).
  4. Navigate to **Build > App Hosting**.
  5. Connect your GitHub account and select the `studio` repository.
  6. Firebase will now automatically deploy your changes whenever you push to GitHub!

## 🚀 Terminal & Mobile Help
- **Terminal Note**: Main ek AI assistant hoon aur aapke liye **terminal commands** (jaise `firebase login`) nahi chala sakta. Yeh commands aapko manually chalaani hongi agar aap PC par hain.
- **Mobile Workflow**: Mobile par terminal ki **bilkul zaroorat nahi hai**. Main saara setup `package.json` ke zariye automatically kar deta hoon.
- **Auto-Sync**: Aapka code preview window mein turant update hota hai.

## 🛠 Tech Stack
- **Next.js 15 (App Router)**
- **Firebase** (Auth, Firestore)
- **Tailwind CSS**
- **ShadCN UI**
- **Genkit** (AI Analysis)

## 📱 Mobile Workflow
1. **No Terminal Needed**: Aapko `firebase login` ya `deploy` ki zaroorat nahi hai.
2. **GitHub Visibility**: Agar GitHub par code nahi dikh raha, toh Source Control tab se "Push" kijiye.
3. **Real-time Updates**: Jo bhi badlav chahiye, mujhe batate rahiye!
